# Majikoi Steam Script Arc Tools

Tools for unpacking, editing, and repacking the **MinatoSoft PAC** archives and **BIN script** files used by *Maji de Watashi ni Koi Shinasai!* (Steam version).

## Scripts

### `minatosoft_pac.py` — PAC archive tool

Handles the proprietary **PAC** format (magic `PAC\x00`). Files are stored back-to-back with optional zlib compression; the file table is Huffman-encoded and XOR'd with `0xFF` at the end of the archive.

```bash
# Unpack an archive
python minatosoft_pac.py unpack Script.pac Extracted/

# Repack a directory back into a PAC
# ⚠️  Must unpack first — repack requires the manifest.json generated by unpack
python minatosoft_pac.py repack Extracted/ Script_new.pac

# Run round-trip self-tests
python minatosoft_pac.py test Script.pac
```

The `manifest.json` written during unpack records filenames (Shift-JIS), compression flags, and offsets so the repack is byte-accurate for unchanged files.

---

### `majikoi_bin_tool.py` — BIN script tool

Parses the script blobs inside each `.bin` file. The script section starts after a fixed 8-byte marker (`1C 00 00 00 00 00 00 00`) and is split at null bytes into typed segments (`TEXT`, `NAME`, `VOICE_DIALOGUE`, `SYSTEM`, `RESOURCE`).

```bash
# Extract a single .bin
python majikoi_bin_tool.py extract IN/act_a_op.bin OUT/act_a_op/

# Extract all .bin files in a directory
python majikoi_bin_tool.py extract-all IN/ OUT/

# Repack after editing
# ⚠️  Must extract first — repack requires *_head.bin and *_manifest.json from extraction
python majikoi_bin_tool.py repack OUT/act_a_op/ patched_act_a_op.bin [--mode preserve|rebuild]

# Repack all
python majikoi_bin_tool.py repack-all OUT/ Patched/ [--mode preserve|rebuild]
```

Each extraction produces:
- `*_head.bin` — binary header up to and including the marker
- `*_script.bin` — raw script blob
- `*_script.txt` — human-editable file with `SEG` blocks (edit the second line of each block)
- `*_manifest.json` — segment index, offsets, and types for repacking

**Repack modes:**
- `preserve` — keeps each segment at its original byte length (pads/truncates); safe for fixed-offset engines
- `rebuild` — allows segments to grow or shrink freely

## Patch workflow (no original files modified)

The included `Majikoi.exe` is a patched build that loads **`Update.pac`** (and `Update1.pac`, `Update2.pac`, … in order) on top of the base archives. This means you never touch `Script.pac` or `Config.pac` directly.

**Full workflow:**

```
Script.pac
    │
    ▼  minatosoft_pac.py unpack
IN/  (raw .bin files)
    │
    ▼  majikoi_bin_tool.py extract-all
OUT/ (editable *_script.txt per file)
    │   ← edit translations here
    ▼  majikoi_bin_tool.py repack-all
Patched/  (translated .bin files)
    │
    ▼  minatosoft_pac.py repack
Update1.pac  ← drop next to Majikoi.exe; no originals overwritten
```

Subsequent patch iterations become `Update2.pac`, `Update3.pac`, and so on.
