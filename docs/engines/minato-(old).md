# Minato (Old)

Engine Minato lawas ini yang ngebawa seri Majikoi. Mereka nyimpen semua isinya di dalam archive berekstensi `.pac`.

Langkah kerjanya gampang: pakai panel sebelah kiri buat unpack file `.pac` tersebut ke dalem folder. Setelah kebongkar semua, kamu bakal nemu file-file `.bin` yang isinya cerita.

Nah, buat file `.bin` ini, kamu beralih ke panel sebelah kanan. Di situ kamu bisa ngekstrak segmen teks (SEG) dari dalam `.bin` supaya jadi file teks yang gampang ditranslate. Setelah kamu puas dengan terjemahannya, pastikan kamu milih 'Repack Mode' yang sesuai sama versi game Majikoi-mu di dropdown menu. Setelah itu, inject lagi teksnya ke `.bin` aslinya, dan terakhir tinggal repack folder tadi jadi file `.pac` yang baru.
