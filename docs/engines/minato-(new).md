# Minato (New)

Engine Minato generasi baru ini sering disebut ACV1, dan dipakai di game-game kayak Wagamama High Spec. Mereka pakai format archive `.dat` yang sayangnya butuh kunci enkripsi khusus (Hex Key) buat dibuka.

Nah, sebelum mulai ngapa-ngapain, pastiin kamu udah set Master Hex Key sama Script Hex Key di menu settingnya. Untungnya, tool ini udah di-set default pakai kunci standar game Minato New. Jadi kalau kamu ngerjain game biasa, nggak usah diganti-ganti, langsung hajar aja unpack file `.dat`-nya. Kamu juga bisa atur tingkat kompresi Zlib-nya kalau mau file jadinya lebih kecil.

Setelah unpack `.dat`, script ceritanya bisa kamu ekstrak jadi dua pilihan: antara file JSON atau CSV. Kalau kamu suka kerja pakai Excel atau Google Sheets, format CSV itu ngebantu banget loh. Begitu beres ngetranslate, tinggal inject balik hasil CSV atau JSON-nya jadi script, terus repack ulang folder kamu ke `.dat`.
