# Malie

Malie Engine ini sering muncul di game terkenal kayak Sharin no Kuni, G-Senjou no Maou, sampai Kami no Rhapsody. File archive mereka (biasanya pakai ekstensi `.dat` atau `.lib`) itu terenkirpsi, makanya kamu harus dekripsi filenya dulu pakai tool ini supaya bisa bongkar isinya. Setelah kamu beres ngedit isinya, jangan lupa dienkripsi ulang ya!

Untuk masalah script cerita, proses penerjemahannya agak unik di Malie. File untuk nama karakter dan dialognya itu terpisah. Pertama, pilih folder tempat kamu naruh script mentah hasil dekripsi. Dari situ, kamu harus nge-ekspor Nama Karakter dan Dialog ceritanya ke dalam file yang beda-beda. 

Setelah kamu kelar translate kedua file tersebut, penting banget buat nginget urutan patch-nya: kamu wajib patch/inject si file Nama Karakter DULU, baru setelah itu file Dialognya. Kalau kebalik atau cuma salah satu, bisa berantakan. Terakhir, kalau game ini pakai file gambar dengan format `.mgf`, kamu bisa pakai fitur convert MGF ke PNG biar bisa diedit dan dibalikin lagi jadi MGF pas udah selesai.
