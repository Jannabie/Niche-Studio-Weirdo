# Minori Engine

> Unpack `.paz` archives and repack folders using `fuckpaz`.

**Example games:** ef - a fairy tale of the two, eden*, Supipara, Mashiro Iro Symphony, Natsuzora no Perseus, and more

---

## Tools Available

### Archive Tools (`.paz`)

Minori uses `.paz` archives indexed by game. You must select the correct **Game Index** from the dropdown.

| Button | Action |
|---|---|
| **Unpack .paz** | Extract archive contents to a folder |
| **Repack Folder** | Pack folder contents back into a `.paz` |

**How to use:**

1. **Select Game Index** from the dropdown — choose the specific game and version you're working with:
   - `ef – the first tale (jp)` → Index 0
   - `ef – the first tale (en)` → Index 1
   - `eden* (jp)` → Index 7
   - `Mashiro Iro Symphony (jp)` → Index 10
   - *(and many more — see the full list in the app)*

2. Browse → select your `.paz` file
3. Browse → select the output folder
4. Click **Unpack .paz**

**To repack:**
1. Select the same Game Index as when you unpacked
2. Browse → select the folder with your modified files
3. Click **Repack Folder** → save the output `.paz`
4. Replace the original in the game data folder

> **Warning:** Always use the exact same Game Index for unpacking and repacking, or the archive will be corrupted.
