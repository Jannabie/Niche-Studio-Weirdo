# Minori

Kalau kamu mau ngerjain game-game garapan Minori kayak ef series, eden*, atau Supipara, tool ini udah support archive mereka yang formatnya `.paz`. 

Ada satu trik super penting di engine Minori: kamu wajib milih "Game Index" yang bener dari dropdown menu sebelum mulai. File `.paz` ini beda-beda cara penanganannya tergantung gamenya apa. Misalnya kalau main eden* yang versi Jepang, kamu harus milih Index 7. Kalau salah index, archive kamu bakal korup dan gamenya mogok.

Setelah kamu milih index yang sesuai, tinggal unpack file `.paz` tersebut jadi sebuah folder. Di dalam foldernya, kamu bisa edit, modif gambar, atau nerjemahin script sebebasnya. Setelah semuanya selesai, dengan index game yang sama persis, kamu tinggal pencet repack buat ngubah folder itu jadi `.paz` lagi.
